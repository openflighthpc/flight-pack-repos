# Flight Pack Repos

A collection of descriptors for Flight Pack repositories for use with
Flight Pack.

## Overview

These descriptors are for use with Flight Pack and provide
orchestration scripts, information and configuration for a number of
commonly used software package collections.

## Installation

This collection should be used with an existing functional
installation of Flight Pack. Please refer to Flight Pack project for
information on how to install.
