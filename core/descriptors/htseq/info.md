# htseq(7) -- High-Throughput Sequencing application pack
{:data-extra="Alces Flight Ltd"}

## CONTENT

This pack includes a number of common applications for high-throughput sequencing:

 * abyss
 * bamtools
 * bcftools
 * bedtools2
 * bowtie
 * bowtie2
 * bwa
 * cufflinks
 * fastqc
 * freebayes
 * gatk
 * hisat2
 * picard
 * plink
 * rsem
 * samtools
 * seqan
 * shapeit
 * sratoolkit
 * star
 * tophat
 * trimgalore
 * vcftools
