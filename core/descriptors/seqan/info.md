# seqan(7) -- Sequence Analysis application pack
{:data-extra="Alces Flight Ltd"}

## CONTENT

This pack includes a number of common applications for sequence analysis:

 * augustus
 * bartender
 * blast
 * blat
 * bwa-mem2
 * exonerate
 * hmmer
 * interproscan
 * mummer
 * muscle
 * ncbi-toolkit
 * trf
 * varscan
 * viennarna
